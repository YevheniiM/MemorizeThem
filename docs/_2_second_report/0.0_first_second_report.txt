Протягом тижня виконувались роботи по знаходженню інформації щодо можливого використання Facebook API.

Були зроблені такі кроки:

1. Затверджена головна ідея проекту.
2. Здійснений пошук потрібного апі.
3. Відсіювання апі, яке не буде задовольнати певним умовам при
   відборі даних, необхідних для роботи додатку.
4. Обране апі - Google Contacts API для отримання інформації
   про друзів та знайомих користувача.
5. Апі було протестовано належним чином.
6. Була створена певна модель даних з якими буде працювати
   додаток.
